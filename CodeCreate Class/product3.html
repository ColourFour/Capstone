CardVerse Gallery

Example for the Interactive Trading Card Gallery project. Replace the theme (players/characters/idols) with something you like.

Card Grid Layout

Collect and show off your favourites.

Grid Profiles Click Effects
  • At least 6 cards arranged in a flexbox or CSS grid layout.
  • Each card has an image, name, and one-line description.
  • Cards all have similar size and style so the grid looks neat.

Click Interaction

Make cards feel alive when you click them.

  • When a card is clicked, use JavaScript to toggle a CSS class.
  • The class could change background color, border, or show “selected”.
  • Optionally, reveal hidden stats or extra info when selected.

Filters & Favourites (Stretch Ideas)

Add some extra polish.

  • Filter buttons at the top (e.g. “Attackers”, “Defenders”, “All”).
  • Clicking a filter shows/hides some cards using JavaScript.
  • Favourite counter: when you click a heart icon on a card, increase a total favourites number displayed on the page.
