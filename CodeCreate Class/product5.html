About Me 2.0

Example for the About Me + Fun-Fact Generator project. This is a model structure you can copy and customise.

Hero Section

Introduce yourself in one screen.

About Me Hero Section Personal
  • Big heading: “Hi, I’m [Your Name]”.
  • Short paragraph on who you are and what you like to do.
  • Optional: a photo or avatar image placed next to the text.

Hobbies & Favorites

Show what you’re into.

  • Use two sections or columns: one for hobbies, one for favourites.
  • Hobbies: sports, music, games, reading, etc.
  • Favourites: favourite game, movie, food, artist, etc.

Random Fun-Fact Button

Make the page interactive.

  • Have a paragraph that says: “Click the button for a random fun fact about me.”
  • Button: “Generate Fun Fact”.
  • JavaScript array with 5–10 fun facts; each click shows one random fact in a text area.

Stretch Ideas

Add more personality.

  • Dark/light mode toggle button that switches a class on the <body>.
  • Simple skill bars (e.g. “HTML █████░░░░ 50%”).
  • Smooth scrolling and hover animations on links and buttons.
