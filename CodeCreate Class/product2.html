PixelPlay Web Arcade

Example for the Web Arcade – Small Game Collection project. Your version will include actual JavaScript games.

Arcade Home

Three tiny games. Infinite procrastination.

Arcade RPS Number Guess
  • Title: “PixelPlay Web Arcade”.
  • Short intro explaining that there are 2–3 games below.
  • Navigation buttons or links: “Play RPS”, “Play Number Guess”, “Play Snake”.

Game 1 – Rock–Paper–Scissors

Classic battle against the computer.

  • Buttons: Rock, Paper, Scissors.
  • JavaScript randomly chooses the computer’s move.
  • Display who won each round and keep a score while the page is open.

Game 2 – Number Guess

Can you guess the hidden number?

  • Input box where the player guesses a number between 1 and 20.
  • JavaScript compares the guess and shows “too high”, “too low”, or “correct!”.
  • Count how many attempts the player needed.

Optional Game 3 – Snake

For extra challenge.

  • Use the provided Snake example or another tiny game.
  • Place it in its own section with instructions and a title.
  • Style each game’s section with different colors so they feel separate.
